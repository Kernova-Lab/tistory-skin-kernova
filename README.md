# Kernova Lab Tistory Skin

Kernova Lab 기술 블로그를 위한 티스토리 스킨 프로젝트입니다.

## 목표

- 긴 기술 글을 읽기 좋은 다크 테마
- 모바일과 데스크톱 모두 안정적인 레이아웃
- 코드 블록, 표, 이미지, 인용문에 최적화된 본문 스타일
- 검색 노출과 광고 배치를 고려한 구조
- 티스토리 스킨 편집기에서 관리하기 쉬운 파일 구성

## 주요 파일

- `skin.html`: 티스토리 화면 구조와 치환자
- `style.css`: 레이아웃, 디자인 토큰, 본문 스타일
- `index.xml`: 스킨 정보와 사용자 설정
- `images/script.js`: 검색, 메뉴, 목차 등 보조 기능
- `preview1600.jpg`, `preview560.jpg`, `preview256.jpg`: 스킨 미리보기 이미지

## 배포

티스토리 스킨 업로드 시 다음 파일을 최상위에 두고 압축합니다.

```text
skin.html
style.css
index.xml
images/
preview1600.jpg
preview560.jpg
preview256.jpg
```

로컬 작업용 자료, 내부 문서, 임시 파일, 압축 산출물은 저장소에 포함하지 않습니다.
